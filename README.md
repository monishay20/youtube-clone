# YouTube Clone 🎥

A YouTube UI clone built with *React + Vite*.  
This project replicates the design of YouTube with modern frontend technologies.

## 🚀 Features
- Video listing page
- Responsive design
- Search bar UI
- Modern styling with CSS/JSX
